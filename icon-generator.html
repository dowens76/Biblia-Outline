Quick Start

This page generates simple icons for the Bible Outline Builder extension.

  1. Click each "Download" button below to save the icons
  2. Create a folder called icons in your extension directory
  3. Save the downloaded files as icon16.png, icon48.png, and icon128.png
  4. Load the extension in Brave/Chrome
